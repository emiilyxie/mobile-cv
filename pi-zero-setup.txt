1. download Raspbian - Buster Light https://www.raspberrypi.org/downloads/raspbian/

2. install image: https://www.raspberrypi.org/documentation/installation/installing-images/README.md

3. resize /boot and /root: to have a large off-line accessible /boot partition
https://learn.adafruit.com/resizing-raspberry-pi-boot-partition#edit-partitions

4. setup time-lapse picam:
sudo apt-get install python3-picamera
https://projects.raspberrypi.org/en/projects/raspberry-pi-zero-time-lapse-cam
https://projects.raspberrypi.org/en/projects/getting-started-with-picamera
http://picamera.readthedocs.io/en/release-1.13/
